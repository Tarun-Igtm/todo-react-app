# 🚀 Modern React Todo App

A modern and responsive Todo Application built using React and Vite.

## ✨ Features

- Add Tasks
- Delete Tasks
- Complete Tasks
- Filter Tasks
- Local Storage
- Responsive Design
- Glassmorphism UI
- Modern Animations

## 🛠 Tech Stack

- React
- Vite
- CSS3
- React Icons

## 📱 Responsive

Works on:
- Desktop
- Tablet
- Mobile
